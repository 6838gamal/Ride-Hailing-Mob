# Ride Hailing App

A Flutter ride-hailing mobile app project.

## Project Overview

This is a Flutter project (`ride_hailing`) originally created with FlutLab. Currently the codebase is a bare "Hello World" starter — the Flutter project structure, Android/iOS/web targets, and pubspec are in place, but the ride-hailing features have not yet been built.

## Stack

- **Framework:** Flutter (Dart)
- **Targets:** Android, iOS, Web
- **Flutter SDK:** ^3.0.0

## Running the App

Flutter is not natively installed in this Replit environment. To run the app you have two options:

1. **Flutter Web on Replit:** Install the Flutter SDK and run `flutter run -d web-server --web-hostname 0.0.0.0 --web-port 5000`
2. **Local development:** Clone the repo and run `flutter run` locally with Flutter installed.
